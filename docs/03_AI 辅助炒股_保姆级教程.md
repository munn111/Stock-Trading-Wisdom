## 二、提问模板（直接复制）

### 1. 获取个股 K 线图
```
帮我画一下 [股票名称/代码] 的 K 线图，要求：
- 显示 5 日、10 日、20 日均线
- 标注成交量
- 标出最近的涨停板
```

### 2. 战法选股
```
帮我找符合以下条件的股票：
- 条件 1：近期有涨停或巨量阳线（涨幅>7%）
- 条件 2：回踩 5 日均线
- 条件 3：成交量缩量至前一日一半以下
请列出股票名称、代码、价格、买入策略。
```

### 3. 板块复盘
```
帮我整理一下 [日期] 的涨停板块分析，要求：
- 列出涨停原因（新闻/政策/事件）
- 按梯队分类（T0 龙头、T1 核心、T2 跟涨）
- 给出操作策略（买入时机、卖出信号）
```

### 4. 策略回测
```
帮我测试一下 [策略名称] 的历史胜率，例如：
- 龙头股首次跌停后次日买入
- 回踩 5 日线缩倍量买入
请给出胜率、平均收益、最大回撤。
```

## 三、如何获取报告文件？

我生成的报告和图片会保存在服务器，您可以通过以下方式获取：

### 方式 1：飞书发送（推荐）
1. 您提供**飞书绑定手机号**。
2. 我在 Mac 上执行 `lark auth login` 登录。
3. 您将登录后的 **Token** 发给我。
4. 我自动将图片/报告发送到您的飞书。

### 方式 2：直接复制
- **文字报告**：我直接输出在对话框，您复制粘贴到 Word/飞书文档。
- **K 线图**：我提供 Base64 编码，您用在线工具转换为图片（或让技术人员帮您下载）。

### 方式 3：服务器下载（需技术人员）
- 文件路径：`/root/GenericAgent/temp/文件名.png`
- 使用 SCP 命令下载：`scp root@服务器 IP:/root/GenericAgent/temp/文件名.png ./`

## 四、常见问题（FAQ）

### Q1：你的数据是实时的吗？
**A**：我的数据来源于**每日收盘后更新**的本地数据库，非实时行情。盘中监控需运行自动化脚本。

### Q2：你的推荐保证赚钱吗？
**A**：**不保证**。我提供的是基于历史数据的**概率分析**，股市有风险，决策需自主。

### Q3：我不懂代码，能运行监控脚本吗？
**A**：可以！您只需让技术人员帮您运行一次脚本，之后它会**自动监控**并在符合条件时输出结果。

### Q4：如何更新数据？
**A**：您只需对我说"更新数据"或"读取最新行情"，我会自动调用工具更新本地数据库。
