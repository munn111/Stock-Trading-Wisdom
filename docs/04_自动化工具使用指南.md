---

## 一、环境准备

### 1. 安装依赖
在服务器或本地电脑执行：
```bash
pip install pandas matplotlib akshare
```

### 2. 下载脚本
脚本位置：`Stock-Trading-Wisdom/scanner_ma5_monitor.py`

---

## 二、运行监控脚本

### 1. 全市场扫描
```bash
python scanner_ma5_monitor.py
```
- **功能**：扫描全市场 4000+ 只股票，找出符合"回踩 5 日线缩倍量"模式的个股。
- **输出**：`target_stocks_ma5_shrink.csv`（含股票名称、代码、价格、买入策略）。

### 2. 监控指定股票
修改脚本中的 `stock_list`：
```python
stock_list = ['001259', '002407', '000666']  # 替换为您想监控的股票
```
然后运行：
```bash
python scanner_ma5_monitor.py
```

## 三、定时任务（可选）

### 1. Linux 服务器
编辑 crontab：
```bash
crontab -e
```
添加每日 15:30 执行（收盘后）：
```bash
30 15 * * * cd /root/GenericAgent/temp && python scanner_ma5_monitor.py
```

### 2. Windows 电脑
使用"任务计划程序"：
- 触发器：每个交易日 15:30。
- 操作：启动程序 `python.exe`，参数 `scanner_ma5_monitor.py`。

## 五、故障排查

### 问题 1：脚本运行报错 `ModuleNotFoundError`
**解决**：安装缺失的库，例如 `pip install akshare`。

### 问题 2：数据更新失败
**解决**：检查网络连接，或手动更新数据（对我说"更新数据"）。

### 问题 3：K 线图无法显示
**解决**：检查 matplotlib 后端设置，或改用服务器生成后下载图片。
