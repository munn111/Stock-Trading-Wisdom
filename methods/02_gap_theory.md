# 缺口理论实战

## 1. 缺口的分类
- **突破缺口**：底部盘整后，放量跳空突破箱体。**策略：立即跟进，目标看高一线。**
- **中继缺口 (测量缺口)**：趋势中途，缩量跳空。**策略：持股待涨，缺口不补不出局。**
- **衰竭缺口**：高位加速后，巨大成交量跳空。**策略：逢高减仓，缺口必补。**

## 2. 实战口诀
- "缺口不补，强势如虎"。
- "三日不补缺，必有一波涨"。
- "高位巨量缺口，见顶信号"。

## 3. 特殊形态：岛形反转
- 底部：下跌缺口 + 横盘 + 上涨缺口 -> 强烈反转信号。
- 顶部：上涨缺口 + 横盘 + 下跌缺口 -> 强烈见顶信号。
