# 市场情绪周期记忆

## 1. 周期阶段
- **冰点期**：涨停<20 家，连板高度<3 板，昨日涨停今日大跌。**策略：空仓或轻仓试错首板。**
- **复苏期**：涨停 30-50 家，出现 3-4 板，亏钱效应减少。**策略：加仓核心股。**
- **高潮期**：涨停>80 家，连板高度>7 板，鸡犬升天。**策略：持股待涨，去弱留强。**
- **退潮期**：高位股集体跌停，核按钮频发，涨停家数骤减。**策略：清仓空仓，管住手。**

## 2. 关键信号

## 3. 周五效应与周一效应
- **周五**：资金博弈周末利好，易出涨停，但也易炸板 (若周末无消息)。
- **周一**：若周末有利好，前排一字买不到；若利空，核按钮集中营。
